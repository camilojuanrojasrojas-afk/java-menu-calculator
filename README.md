# 🧮 Java Menu Calculator

## 🚀 Overview
Console-based Java application that provides a menu-driven interface to perform multiple mathematical operations and functionalities.

---

## 💡 Features
- Addition, subtraction, multiplication, division  
- Interactive menu system  
- User input handling  
- Logical flow control  

---

## 🛠️ Tech Stack
- Java  
- NetBeans  

---

## ▶️ How to Run
1. Open in NetBeans  
2. Run the main class  
3. Select options from the menu  

---

## 🎯 Purpose
This project focuses on improving logical thinking, user interaction, and control structures in Java.
